# AI Agent Task 폴더

> AI 에이전트에게 작업을 지시하기 위한 태스크 관리 폴더

---

## ? 폴더 구조

```
ai_agent_task/
├── README.md              ← 현재 문서
├── _완료모음/              ← 완료된 작업 아카이브
│   └── [완료된 작업 폴더들]/
├── [작업명1]/
│   └── [작업명1]_계획.md
├── [작업명2]/
│   └── [작업명2]_계획.md
└── ...
```

---

## ? 사용 목적

### 1. 작업 지시서 작성
- 각 폴더는 **하나의 작업 단위**를 나타냄
- 폴더 내 `.md` 파일에 AI 에이전트가 수행할 **작업 내용을 상세히 기술**
- 작업 범위, 목표, 제약 조건, 참고 파일 등을 명시

### 2. AI 에이전트 활용
- AI 에이전트에게 해당 `.md` 파일을 컨텍스트로 제공
- 에이전트가 지시서를 읽고 작업 수행
- 복잡한 작업을 단계별로 나눠서 진행 가능

### 3. 완료 작업 관리
- 완료된 작업은 `_완료모음/` 폴더로 이동
- 작업 히스토리 및 레퍼런스로 활용

---

## ? 작업 지시서 작성 가이드

### 권장 파일명
```
[작업명]_계획.md
[작업명]_분석.md
[작업명]_가이드.md
```

### 권장 구조
```markdown
# [작업명]

## 목표
- 이 작업이 달성해야 할 것

## 범위
- 수정할 파일/폴더
- 영향받는 시스템

## 작업 단계
1. 첫 번째 단계
2. 두 번째 단계
3. ...

## 제약 조건
- 지켜야 할 규칙
- 건드리면 안 되는 것

## 참고 자료
- 관련 문서 경로
- 레퍼런스 링크

## 완료 조건
- 작업이 완료되었다고 판단할 기준
```

## ? 팁

1. **작업 분리**: 큰 작업은 여러 개의 `.md` 파일로 분리
2. **단계별 진행**: 한 번에 모든 것을 하지 말고 단계별로 진행
3. **명확한 지시**: 모호한 표현 대신 구체적인 파일 경로와 코드 예시 포함
4. **완료 이동**: 작업 완료 시 폴더째로 `_완료모음/`으로 이동
