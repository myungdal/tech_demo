# 게임서버선택 기능

## 목표
- 유저가 접속할 게임 서버를 선택하는 UI 및 로직 구현
- 서버 상태(온라인/오프라인/점검) 표시
- 서버별 유저 수/혼잡도 표시

---

## 범위

### 클라이언트
- 서버 목록 UI
- 서버 상태 조회 API 호출
- 서버 선택 후 접속 처리

### 서버
- 서버 목록 조회 API
- 서버 상태 관리
- 로드밸런싱 정보 제공

---

## 작업 단계

1. **서버 목록 API 설계**
   - 서버 ID, 이름, 상태, 유저 수 반환
   - 점검 예정 시간 포함

2. **클라이언트 UI 구현**
   - 서버 목록 표시
   - 상태별 색상 구분 (녹색=쾌적, 노랑=혼잡, 빨강=포화)
   - 추천 서버 표시

3. **서버 선택 로직**
   - 선택한 서버로 접속 요청
   - 접속 실패 시 재시도/다른 서버 안내

4. **최근 접속 서버 기억**
   - 로컬 저장소에 마지막 접속 서버 저장
   - 다음 접속 시 자동 선택

---

## 제약 조건
- 점검 중인 서버는 선택 불가 (비활성화)
- 서버 상태는 10초마다 갱신
- 접속 중 서버 변경 불가
