# 인벤토리 시스템

## 목표
- 카드, 장비, 재화, 소모품 통합 인벤토리
- 정렬/필터링/검색 기능
- 아이템 사용/장착/분해/강화 연동

---

## 범위

### 서버
- 인벤토리 데이터 관리 (DB)
- 아이템 추가/삭제/이동 API
- 인벤토리 용량 관리

### 클라이언트
- 인벤토리 UI
- 드래그&드롭 정렬
- 필터/검색 기능

---

## 작업 단계

1. **DB 스키마 설계**
   - 유저별 인벤토리 테이블
   - 아이템 종류 (카드/장비/재화/소모품)
   - 슬롯 위치, 수량, 잠금 상태

2. **인벤토리 API**
   - 조회: 전체/카테고리별
   - 추가: 아이템 획득
   - 삭제: 아이템 소모/분해
   - 이동: 슬롯 정렬

3. **클라이언트 UI**
   - 탭 구분 (카드/장비/재화/기타)
   - 아이템 상세 팝업
   - 정렬 (등급/이름/획득순)
   - 필터 (등급/속성)

4. **연동 기능**
   - 장비 장착/해제
   - 카드 덱 편성
   - 아이템 사용
   - 강화/합성 연결

---

## 제약 조건
- 인벤토리 용량 제한 (기본 100, 확장 가능)
- 귀속 아이템 거래 불가
- 동시 조작 방지 (락 처리)
