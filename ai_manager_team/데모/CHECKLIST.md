## 🎤 면접 예상 질문 & 답변

### Q1: AI가 작성한 코드를 어떻게 신뢰하나요?

```
A: "AI 코드는 반드시 리뷰합니다. 체크리스트가 있습니다:
   - 컴파일 가능 여부
   - 기존 컨벤션 준수
   - 스레드 안전성
   - 에러 처리
   - 테스트 커버리지
   
   AI는 초안 작성자이고, 최종 책임은 개발자에게 있습니다."
```

### Q2: AI 없이도 같은 속도로 개발할 수 있나요?

```
A: "AI 없이도 개발 가능하지만 속도 차이가 있습니다.
   20년 경험으로 패턴을 알고 있어서 AI 결과물을 빠르게 검증할 수 있습니다.
   AI는 타이핑과 보일러플레이트를 줄여주는 도구입니다."
```

### Q3: AI 도구 비용 대비 효과는?

```
A: "Cursor Pro 월 $20, Claude Pro 월 $20.
   개발 시간 50% 단축으로 계산하면 ROI는 압도적입니다.
   시니어 개발자 인건비 대비 1% 미만의 비용으로 생산성 2배."
```

### Q4: 팀에서 AI 도입 시 고려사항은?

```
A: "세 가지를 고려합니다:
   1. 보안: 민감한 코드/데이터 AI에 전달 금지 정책
   2. 표준화: .cursorrules 같은 프로젝트 규칙 통일
   3. 교육: AI 프롬프트 작성법, 리뷰 체크리스트 공유"
```

### Q5: 가장 효과적이었던 AI 활용 사례는?

```
A: "서버 이전/통합 작업에서 수백 개 설정 파일 마이그레이션.
   패턴 파악 후 AI에게 변환 규칙 설명.
   수작업 2주 예상 → 2일 완료.
   오류율도 수작업보다 낮았습니다."
```
