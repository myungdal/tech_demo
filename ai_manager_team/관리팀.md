## 프로젝트 개요

**Nearest3**는 C++ MMO 게임 서버 + Unreal Engine 5 클라이언트 프로젝트입니다.

| 영역 | 기술 |
|------|------|
| 서버 | C++20, Windows IOCP, MSSQL |
| 클라이언트 | Unreal Engine 5.6 |
| 코드 생성 | TypeScript (`setup/`) |

---

## 필수 규칙 문서

프로젝트 작업 전 반드시 숙지해야 할 규칙들입니다.

| 우선순위 | 문서 | 핵심 내용 |
|----------|------|-----------|
| **필수** | `AGENTS.md` | 전체 규칙 요약 (자동생성됨) |
| **필수** | `.cursor/rules/*.mdc` | 상세 규칙 원본 |

### 절대 금지 사항 (Critical)

1. **autoGenerated 파일 수정 금지** - `resource/*.xlsx` 수정 후 재생성
2. **패킷 Writer SetValues 누락 금지** - 쓰레기 데이터 전송됨
3. **`import ServerEngine;` 금지** - C1605 에러 (obj 4GB 초과)

## 폴더 구조

```
nearest3/
├── server/           # 게임 서버 (C++20)
├── client/           # 게임 클라이언트 (UE5)
├── setup/            # 코드 자동 생성 (TypeScript)
├── resource/         # 기획 데이터 원본 (xlsx, sql)
├── bat/              # 빌드/생성 스크립트
├── .cursor/rules/    # AI 규칙 원본 (mdc)
├── .opencode/skill/  # OpenCode Skills (자동생성)
└── ai_*_team/        # AI 협업 폴더
```

## 주요 명령어

### 코드 생성
```batch
bat/generate_all.bat      # 전체 생성 (DB + 서버 + 클라)
bat/generate_server.bat   # 서버만
bat/generate_client.bat   # 클라이언트만
```

### AI 규칙 동기화
```bash
# mdc 파일 수정 후 실행 (CLAUDE.md, AGENTS.md, Skills 재생성)
powershell -ExecutionPolicy Bypass -File "./power_shell/sync_ai_rules.ps1"
```

### 인코딩 수정
```powershell
.\power_shell\check_encoding_full.ps1 -FixMode
```

### 세션 관리
```batch
show_recent_sessions.bat   # 세션 제목 관리용
```

## OpenCode Skills

상황에 맞는 skill을 로드하면 상세 규칙을 확인할 수 있습니다.

| Skill | 사용 시점 |
|-------|-----------|
| `packet-writer` | 패킷 송신 코드 작성 시 |
| `cpp-module` | C++20 모듈 import 작업 시 |
| `code-gen` | 자동생성 파일 관련 질문 시 |
| `server-handler` | 서버 핸들러 작성 시 |
| `client-code` | 클라이언트 공유 라이브러리 작업 시 |
| `unreal-code` | UE5 클라이언트 코드 작성 시 |
| `design-docs` | 기획 문서 작성 시 |
| `ai-tools` | AI 협업 폴더 스크립트 작업 시 |
