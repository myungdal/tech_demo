화면의 4개 코너와 하단을 활용하여 시야를 최대한 확보하는 HUD 스타일 채택.

> UI는 플레이어 입력이 제한된 게임(자동 전투/간접 조작)인 만큼,
> “현재 상태(State)”와 “다음 액션 버튼”이 항상 명확하게 보이도록 구성한다.
> State 정의는 `10_멀티플레이_필드맵/01_유닛_소환_시스템/01_소환_프로세스.md` 참고.

## 12.1 좌상단 (TOP-LEFT)
- **미니맵 (Mini-map)**:
  - 전장 전체 상황 식별
  - 아군/적군/성 위치 표시
  - 클릭 시 해당 지점으로 카메라 이동

## 12.2 우상단 (TOP-RIGHT)
- **관리 메뉴 (System Menu)**:
  - 설정 (환경설정, 사운드 등)
  - 우편함/정산함 (시스템 보상 등)
  - 랭킹 / 클랜 정보
  - 인벤토리 / 덱 편집 (비전투 시)
  
> (레거시) 세금 시스템은 폐기되었으므로, “세금 수령/우편함” 관련 UI는 캐슬 버프/점령 UI로 대체한다.

## 12.3 좌하단 (BOTTOM-LEFT)
- **채팅창 (Communication)**:
  - 전체 / 클랜 / 귓속말 탭 구분
  - 시스템 메시지 (킬/데스 로그, 공성 알림)
  - 투명도 조절로 전장 시야 방해 최소화

## 12.4 하단 중앙 (BOTTOM-CENTER)
- **카드 핸드 (Card Hand)**:
  - 현재 보유한 스킬/유닛 카드 (3~4장)
  - 드래그 앤 드롭으로 사용

## 12.5 우하단 (BOTTOM-RIGHT) - 액션 존
- **메인 액션 버튼 (Context Action)**:
  - **평시**: [소환 확정]
  - **전투 중**: [귀환 명령] (가장 크고 눈에 띄게 배치)
  - **특수**: [공성 시작] 등
- **보조 버튼**:
  - [내 유닛 중점(Focus Unit)] 카메라 복귀
  - [타겟팅 고정/해제]
