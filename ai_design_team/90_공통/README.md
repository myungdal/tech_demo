## 🔑 핵심 개념

### 카드 시스템
- **유닛 카드**: 필드에서 유닛 그룹 소환에 사용
- **개입 카드**: 전투 중 사용하는 스킬/버프/디버프
- 던전에서 획득 → 필드에서 사용

### 밸런스 보호
- 무과금 유저도 최상위 콘텐츠 도달 가능
- 결제 = 시간 단축 (Pay-to-Win 금지)

### 리소스 규격
- 모든 스프라이트: PNG Alpha, 투명 배경
- 유닛: 오른쪽 방향 기준, 코드에서 Flip
- 이펙트: 유닛 애니메이션과 분리 관리
