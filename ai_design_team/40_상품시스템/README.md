## 🔑 핵심 설계 원칙

| 원칙 | 설명 |
|------|------|
| **P2W 방지** | 결제 = 시간 단축, 무과금도 최상위 가능 |
| **적정 가격** | 타 게임 대비 합리적 가격대 |
| **가치 명확성** | 상품의 가치가 명확히 전달 |
| **법적 준수** | 확률 공개, 환불 정책 준수 |
