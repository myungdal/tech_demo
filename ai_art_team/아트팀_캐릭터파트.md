## 스크립트 사용법

### 1. 배경 정리 (이미 완료)
```bash
cd ai_art_team/scripts/character
node clean_background.js    # 12characters.png → 12characters_clean.png
node fix_magenta.js         # 마젠타 추가 제거
```

### 2. 캐릭터 추출 (이미 완료)
```bash
node extract_characters.js  # 12characters_clean.png → individual/01~12.png
```

### 3. 그리드 재조립 (이미 완료)
```bash
node assemble_grid.js       # individual/*.png → 12characters_grid.png
```

## 검토 중인 아이디어: 3D 파이프라인

> **상태**: 아이디어 단계, 미착수

### 접근법

```
2D 픽셀아트 (64x64)
    ↓ AI Image-to-3D
3D 모델 (.glb/.fbx)
    ↓ 애니메이션 적용
애니메이션된 3D
    ↓ Blender CLI 렌더링
스프라이트 시트
```

### 조사된 서비스

**Image-to-3D API:**

| 서비스 | 가격 | API | 비고 |
|--------|------|-----|------|
| Meshy | $0.025-0.15/개 | REST + Python SDK | |
| Tripo3D | $0.05-0.20/개 | REST API | |
| Rodin | $0.40/개 | fal.ai | |

**3D 애니메이션:**

| 서비스 | 가격 | API | 비고 |
|--------|------|-----|------|
| Mixamo | 무료 | ❌ 수동만 | 완전 자동화 불가 |
| Rokoko Vision | $20/월 | REST | 자동화 가능 |
| Move.ai | $0.012/초 | REST | 자동화 가능 |

**렌더링:**
- Blender CLI: `blender -b model.blend -o //sprites/frame_##### -F PNG -a`
- 방향: 오른쪽(→) 1방향

### 주요 발견
- Mixamo는 2D→3D 기능 없음 (3D 모델 필요)
- Mixamo API 없음 → 완전 자동화하려면 유료 서비스 필요
