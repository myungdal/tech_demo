# UI 리소스

게임 인터페이스 관련 아트 리소스.

---

## 예정 리소스

| 카테고리 | 설명 |
|----------|------|
| 공통 UI | 버튼, 팝업, 프레임, 아이콘 |
| HUD | 체력바, 미니맵, 스킬 슬롯 |
| 상점 | 상품 프레임, 가격표, 배너 |
| 카드 | 카드 프레임, 등급 테두리, 속성 아이콘 |
| 메뉴 | 로비, 설정, 인벤토리 |

---

## 제작 가이드라인

1. **해상도**: 1920x1080 기준, 다해상도 대응
2. **스타일**: 판타지 + 미니멀 (과도한 장식 지양)
3. **컬러**: 다크 테마 기반, 등급별 강조색 구분
4. **폰트**: 가독성 우선, 한글/영문 지원
