# 배경 리소스

맵, 타일, 환경 오브젝트 등 배경 아트 리소스.

---

## 예정 리소스

| 카테고리 | 설명 |
|----------|------|
| 필드맵 타일 | 풀, 흙, 돌, 물 등 기본 타일 |
| 던전맵 타일 | 던전 바닥, 벽, 함정 |
| 오브젝트 | 나무, 바위, 건물, 장식물 |
| 성/건물 | 성벽, 탑, 문, 내부 |
| 환경 | 날씨, 시간대별 조명 |

---

## 맵 종류

### 필드맵 (MMO 전장)
| 지역 | 테마 | 특징 |
|------|------|------|
| 초원 | 녹색 평원 | 기본 필드, 시야 넓음 |
| 숲 | 울창한 숲 | 시야 제한, 매복 유리 |
| 사막 | 모래 지형 | 이동속도 감소 |
| 설원 | 눈/얼음 | 빙결 효과 증가 |
| 화산 | 용암 지대 | 화염 데미지 증가 |

### 던전맵 (싱글/코옵)
| 던전 | 테마 | 보스 |
|------|------|------|
| 고블린 동굴 | 동굴 | 고블린 킹 |
| 언데드 묘지 | 무덤 | 리치 |
| 드래곤 둥지 | 화산 | 드래곤 |
| 심연 | 암흑 | 마왕 |

---

## 타일 규격

- 기본 타일: 64x64 픽셀
- 대형 오브젝트: 128x128 이상
- 스타일: 2D 탑뷰, 밝은 판타지풍
